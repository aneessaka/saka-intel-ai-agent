# ⚙️ Workflow Overview – Saka Intel Agent

## 🔗 Inputs
- Company URL
- LinkedIn URL

## 🧠 Flow
1. Company URL is crawled via Firecrawl → summarized by LLM
2. LinkedIn profile is analyzed for role, hiring influence, pain points
3. LLM combines both into a structured pre-call brief

## ✅ Output
- Company Snapshot
- Prospect Summary
- Strategic Talking Points
- Objection Handling
- Rapport Starters
